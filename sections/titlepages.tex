	% The abstract
	\arinc{} addresses the increasing number of
	microcontroller units in aviation systems.
	By unifying applications under
	a single API, and using time and space partitioning,
	it is possible to use a single computing
	device to support multiple safety critical applications.
	This project describes the implementation of the \arinc{} standard,
	as an Operating System,
	on an embedded platform, Mini-M4 for STM32.
