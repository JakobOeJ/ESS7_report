	% The abstract
	ARINC 653 addresses the increasing number of microcontroller units in aviation systems. By partitioning and unifying applications under a single API, it is possible to use a single computing device to support multiple safety critical applications. This projects aim is to implement part of the 653 standard, to position future research into applying this standard to other industries.
